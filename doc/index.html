Como criar um pendrive bootável

Usando o Win32 Disk Imager no Windows:
1. Baixe o Win32 Disk Imager do site oficial: sourceforge.net
2. Converta a ISO do Quieux Linux para o formato IMG, se necessário (pode-se usar ferramentas como o AnyToISO ou outros conversores online).
3. Conecte o pendrive ao seu computador.
4. Abra o Win32 Disk Imager. Ele deve detectar automaticamente seu pendrive.
5. Selecione o arquivo .img correspondente ao Quieux Linux.
6. Confirme a letra da unidade do pendrive e clique em "Write".
7. Aguarde o processo ser concluído. Uma mensagem de sucesso aparecerá quando estiver pronto.

Usando o Terminal no Linux com o comando DD:
1. Conecte o pendrive ao seu computador.
2. Identifique o dispositivo do pendrive com o comando: sudo fdisk -l. Ele será algo como /dev/sdX.
3. Certifique-se de que o dispositivo está desmontado: sudo umount /dev/sdX*.
4. Use o comando DD para criar o pendrive bootável: sudo dd if=/caminho/para/quieux.iso of=/dev/sdX bs=4M status=progress.
5. Aguarde o processo ser concluído. Quando terminar, o terminal voltará ao prompt.

Como instalar programas

Atualizar a lista de repositórios:
sudo quel -u

Instalar pacotes .quel localmente:
sudo quel -i *caminho do programa*

Buscar o pacote no repositório e realizar o download para instalação:
sudo quel -d *nome do programa*

Buscar o pacote no repositório público e realizar o download para instalação:
sudo quel -pub *nome do programa*
