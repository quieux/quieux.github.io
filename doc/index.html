Como criar um pendrive bootável

Usando o Rufus no Windows:
1. Baixe o Rufus do site oficial: rufus.ie
2. Conecte o pendrive ao seu computador.
3. Abra o Rufus. Ele deve detectar automaticamente o seu pendrive.
4. Em "Seleção de boot", clique em "Selecionar" e escolha a ISO do Quieux Linux.
5. Deixe as outras opções como padrão e clique em "Iniciar".
6. Aguarde o processo ser concluído. Uma mensagem de sucesso aparecerá quando estiver pronto.

Usando o Terminal no Linux com o comando DD:
1. Conecte o pendrive ao seu computador.
2. Identifique o dispositivo do pendrive com o comando: sudo fdisk -l. Ele será algo como /dev/sdX.
3. Certifique-se de que o dispositivo está desmontado: sudo umount /dev/sdX*.
4. Use o comando DD para criar o pendrive bootável: sudo dd if=/caminho/para/quieux.iso of=/dev/sdX bs=4M status=progress.
5. Aguarde o processo ser concluído. Quando terminar, o terminal voltará ao prompt.

Como instalar programas

Instalar pacotes .quel localmente:
sudo quel -i *caminho do programa*

Remover programas instalados no seu computador:
sudo quel -r *caminho do programa*

Atualizar a lista de repositórios:
sudo quel -u

Buscar o pacote no repositório e realizar o download para instalação:
sudo quel -d *nome do programa*

Buscar o pacote no repositório público e realizar o download para instalação:
sudo quel -pub *nome do programa*

Como instalar o Quieux Linux permanentemente

Usando o Refracta Installer:
1. Baixe a ISO do Quieux Linux e crie um pendrive bootável seguindo as instruções acima.
2. Dê boot no computador a partir do pendrive.
3. No ambiente live, abra um terminal e execute o Refracta Installer: sudo refractainstaller.
4. Siga as instruções na tela:
  a. Selecione a opção "Install to partition" e escolha a partição desejada.
  b. Configure as opções de partição conforme necessário.
  c. Siga os prompts para configurar o GRUB e outros ajustes.
5. Quando a instalação estiver completa, reinicie o computador e remova o pendrive.
6. O Quieux Linux deve iniciar a partir do disco rígido.
