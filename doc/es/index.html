Cómo crear un pendrive bootable

Usando Rufus en Windows:
1. Descargue Rufus del sitio oficial: rufus.ie
2. Conecte el pendrive a su computadora.
3. Abra Rufus. Debería detectar automáticamente su pendrive.
4. En "Selección de arranque", haga clic en "Seleccionar" y elija la ISO de Quieux Linux.
5. Deje las otras opciones por defecto y haga clic en "Iniciar".
6. Espere a que el proceso se complete. Aparecerá un mensaje de éxito cuando esté listo.

Usando el Terminal en Linux con el comando DD:
1. Conecte el pendrive a su computadora.
2. Identifique el dispositivo del pendrive con el comando: sudo fdisk -l. Será algo como /dev/sdX.
3. Asegúrese de que el dispositivo esté desmontado: sudo umount /dev/sdX*.
4. Use el comando DD para crear el pendrive bootable: sudo dd if=/ruta/al/quieux.iso of=/dev/sdX bs=4M status=progress.
5. Espere a que el proceso se complete. Cuando termine, el terminal volverá al prompt.

Cómo instalar programas

Instalar paquetes .quel localmente:
sudo quel -i *ruta del programa*

Eliminar programas instalados en su computadora:
sudo quel -r *ruta del programa*

Actualizar la lista de repositorios:
sudo quel -u

Buscar el paquete en el repositorio y realizar la descarga para la instalación:
sudo quel -d *nombre del programa*

Buscar el paquete en el repositorio público y realizar la descarga para la instalación:
sudo quel -pub *nombre del programa*

Cómo instalar Quieux Linux permanentemente

Usando el Refracta Installer:
1. Descargue la ISO de Quieux Linux y cree un pendrive bootable siguiendo las instrucciones anteriores.
2. Arranque la computadora desde el pendrive.
3. En el entorno live, abra un terminal y ejecute el Refracta Installer: sudo refractainstaller.
4. Siga las instrucciones en pantalla:
  a. Seleccione la opción "Install to partition" y elija la partición deseada.
  b. Configure las opciones de partición según sea necesario.
  c. Siga los prompts para configurar el GRUB y otros ajustes.
5. Cuando la instalación esté completa, reinicie la computadora y retire el pendrive.
6. Quieux Linux debería iniciarse desde el disco duro.
